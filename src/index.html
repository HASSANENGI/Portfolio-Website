AI-Powered Attendance System

Revolutionizing attendance tracking, I developed an advanced Face Recognition Attendance System. This cutting-edge solution harnesses YOLOv8 for lightning-fast, accurate face detection in real-time. Once detected, a unique numerical "embedding" is generated from each face, representing its distinct features. This embedding is then instantly compared against a robust database of pre-registered user embeddings. If a precise match is found, the individual is swiftly identified, and their attendance is seamlessly recorded. My proactive approach ensured the integration of complex models, optimizing for unparalleled real-time accuracy and robust identification, guaranteeing reliable attendance tracking. This project profoundly reinforced my skills in Computer Vision, Machine Learning, Python, and efficient data management.

Smart Waste Segregation Robot

Manual trash collection is inefficient, hazardous, and not scalable for large public or industrial spaces. To solve this, I developed an autonomous UGV that detects, collects, and sorts trash without human intervention. The system uses a YOLOv8 model (trained via Roboflow) and an OAK-D camera for real-time trash detection and depth perception. I designed and 3D-printed a 5-DOF robotic arm for picking and binning waste, and integrated ESP32-controlled DC motors for movement. Using Raspberry Pi and ROS, I coordinated all components for perception, control, and navigation. This solution reduces human exposure to waste, increases collection efficiency, and enables smart, scalable waste management.

Realtime Voice Chatbot

This AI Voice Chatbot is a sophisticated real-time communication system designed for seamless, natural conversations. It intelligently captures user voice input in dynamic chunks, delineated by pauses in speech, which are then processed by OpenAI's Whisper for highly accurate Speech-to-Text (STT) transcription. The resulting text is fed to a ChatGPT API, generating an AI-powered conversational response to the user's query. This AI-generated text is then transformed back into natural-sounding speech using Eleven Labs' advanced Text-to-Speech (TTS) capabilities. The entire robust architecture is hosted on a FastAPI server, ensuring efficient and scalable operations, while the user interaction and voice transmission are managed via an Asterisk server, providing a complete, integrated voice chatbot experience.
